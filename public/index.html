Request Header Parser Microservice

Description

This microservice returns information about your request headers including your IP address, preferred language, and software information.

Usage

Make a GET request to the following endpoint:

GET /api/whoami
Try it now!

Example Response

{ "ipaddress": "159.20.14.100", "language": "en-US,en;q=0.9", "software": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36..." }

What You'll Get

ipaddress: Your IP address

language: Your preferred language(s) from Accept-Language header

software: Your browser/software information from User-Agent header
