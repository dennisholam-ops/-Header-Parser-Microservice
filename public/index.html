Request Header Parser Microservice

Example Usage:

[base url]/api/whoami

Example Output:

{"ipaddress":"159.20.14.100","language":"en-US,en;q=0.5","software":"Mozilla/5.0 (X11; Ubuntu; Linux x86_64; rv:50.0) Gecko/20100101 Firefox/50.0"}

What it does:

  • Returns your IP address in the ipaddress field
  • Returns your preferred language in the language field
  • Returns your software/browser in the software field
