Request Header Parser Microservice

Example Usage:

/api/whoami

Example Output:

{"ipaddress":"159.20.14.100","language":"en-US,en;q=0.5","software":"Mozilla/5.0 (X11; Ubuntu; Linux x86_64; rv:50.0) Gecko/20100101 Firefox/50.0"}

What it returns:

  • ipaddress: Your IP address
  • language: Your preferred language from Accept-Language header
  • software: Your browser/software from User-Agent header

API Endpoint:

GET /api/whoami
